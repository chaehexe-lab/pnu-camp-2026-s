# 마지막 열차의 거짓말

AI 용의자 심문 게임 콘셉트의 브라우저 기반 MVP입니다. 사건 속 세 명의 용의자를 심문하고, 증거를 제시해 모순을 찾아낸 뒤 범인을 지목합니다.

## 실행

별도 설치 없이 `index.html`을 브라우저에서 열면 됩니다. 로컬 서버를 사용할 경우:

```powershell
python -m http.server 8000
```

이후 `http://localhost:8000`에 접속합니다.

## 구현 기능

- 5분 제한 추리 플레이
- 용의자 3명 전환 및 자유 질문
- 키워드 기반 답변과 동요 지수
- 증거 5개 제시 및 모순 발견
- 형사의 단서 메모
- 범인 지목과 결과 화면
- 모바일 반응형 레이아웃

현재 버전은 API 키 없이 시연할 수 있도록 키워드 기반 로컬 대화 엔진을 사용합니다. 모든 질문에 용의자가 반응하며, 시간·장소·피해자·소지품·출입 카드·정전에 관한 질문에는 각자의 설정에 맞는 답변을 제공합니다. 향후 LLM API를 연결하면 자유 대화와 장기 기억을 강화할 수 있습니다.
